Abstract

Our project enhances gaming by integrating AI and AR to create immersive, interactive experiences. AI-driven NPCs, powered by reinforcement learning and neural networks, adapt in real-time for dynamic gameplay. AR technology, using spatial mapping and tools like ARCore or ARKit, blends virtual and real-world elements seamlessly. Procedural generation customizes game levels based on player performance, while real-time analytics refine gameplay. Built with Unity and AI frameworks like TensorFlow and PyTorch, this project redefines interactive entertainment through realism, personalization, and innovation.
