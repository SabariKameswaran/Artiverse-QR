Abstract

Our project creates an interactive 3D art gallery using Three.js, a WebGL-based JavaScript library. Users can explore and interact with animated 3D art pieces in real-time. The gallery features scene creation, camera configuration, and renderer optimization to ensure realism. Users can adjust parameters with a GUI debugger and navigate using intuitive controls. The project integrates material and geometry creation for customizable visuals and animation for enhanced interactivity. Built with Three.js and tools like OrbitControls and dat.GUI, it demonstrates how modern web technologies can redefine digital art presentation.
